Установка Vue.js:
В терминале:
npm create vue@latest
npm i
--------------------------------------

Запуск режима разработчика (c запуском сервера) 
npm run dev
--------------------------------------

Запуск сборки проекта (без запуска сервера, только финальная сборка)
npm run build
--------------------------------------

Запуск просмотра собранного проекта 
npm run preview
